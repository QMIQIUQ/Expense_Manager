## 1. 信用卡管理

### 功能

- ✅ 添加多张信用卡
- ✅ 设置账单日和还款日
- ✅ 追踪每张卡的消费
- ✅ 查看账单周期内消费

### 信用卡字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 卡片名称 | 文字 | ✅ | 如 "VISA 尾号1234" |
| 银行 | 文字 | ❌ | 发卡银行 |
| 账单日 | 数字 | ❌ | 每月账单日 (1-28) |
| 还款日 | 数字 | ❌ | 每月还款日 (1-28) |
| 信用额度 | 数字 | ❌ | 信用额度 |

### 账单周期计算

系统根据账单日自动计算当前账单周期的消费：

```
账单日: 15日
当前日期: 12月10日
计算范围: 11月16日 - 12月15日
```

### 使用流程

1. 添加信用卡：Payment Methods → Cards → 添加
2. 记录支出时选择对应信用卡
3. 在 Dashboard 查看各卡消费统计

## 2. 电子钱包

### 功能

- ✅ 管理多个电子钱包
- ✅ 追踪钱包余额
- ✅ 记录充值/消费
- ✅ 查看交易历史

### 电子钱包字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 钱包名称 | 文字 | ✅ | 如 "街口支付" |
| 当前余额 | 数字 | ✅ | 初始余额 |
| 备注 | 文字 | ❌ | 额外说明 |

### 余额追踪

#### 自动扣款

当记录支出并选择电子钱包付款时：
- 系统自动从钱包余额中扣除金额
- 余额不足时显示警告（但仍可记录）

#### 手动充值

1. 进入 Payment Methods → E-Wallets
2. 选择钱包 → 充值
3. 输入充值金额
4. 可选：同时创建收入记录

### 支持的电子钱包

| 钱包 | 说明 |
|------|------|
| 街口支付 | Jkopay |
| Line Pay | Line Pay |
| 悠遊付 | Easy Wallet |
| Pi 拍錢包 | Pi Wallet |
| 全支付 | PX Pay |
| 其他 | 自定义 |

## 3. 支付方式设置

### 访问路径

```
汉堡菜单 → Features → Payment Methods
```

或

```
标签栏 → Payment Methods
```

### 支付方式类型

| 类型 | 追踪方式 |
|------|---------|
| 信用卡 | 账单周期消费 |
| 电子钱包 | 即时余额 |
| 现金 | 无追踪 |
| 银行转账 | 无追踪 |

### 默认支付方式

可在设置中选择默认支付方式，新支出自动使用此方式。

## 4. 使用技巧

### 信用卡账单管理

1. **设置账单日** - 与实际信用卡账单日一致
2. **设置还款日** - 提前几天设置提醒
3. **定期检查** - 每月核对消费记录

### 电子钱包最佳实践

1. **定期对账** - 与实际钱包余额核对
2. **及时充值** - 低余额时充值
3. **分类消费** - 不同用途用不同钱包

### 多卡管理

如果有多张信用卡：
1. 按用途分类（日常消费卡、大额消费卡）
2. 设置不同的消费提醒
3. 定期查看各卡消费比例

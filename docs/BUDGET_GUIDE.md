## 1. 预算设置

### 功能

- ✅ 按分类设置预算
- ✅ 设置月度预算金额
- ✅ 查看预算使用进度
- ✅ 超支提醒

### 预算字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 分类 | 选择 | ✅ | 预算对应的分类 |
| 金额 | 数字 | ✅ | 每月预算金额 |
| 月结日 | 数字 | ❌ | 自定义月结日 |
| 启用滚存 | 开关 | ❌ | 是否滚存剩余预算 |

### 创建预算

1. 进入 Budgets 标签页
2. 点击「添加预算」
3. 选择分类
4. 输入预算金额
5. （可选）设置月结日
6. （可选）启用滚存

## 2. 月结日功能

### 什么是月结日？

月结日决定预算周期的起止日期，方便配合信用卡账单周期。

### 示例

| 月结日 | 本月周期 |
|--------|---------|
| 1日 | 12月1日 - 12月31日 |
| 15日 | 11月16日 - 12月15日 |
| 25日 | 11月26日 - 12月25日 |

### 设置方法

1. 创建/编辑预算
2. 选择「自定义月结日」
3. 选择日期 (1-28)

**注意**: 为避免不同月份天数差异，月结日限制在 1-28 日。

### 与信用卡配合

如果信用卡账单日是每月 15 日：
1. 设置预算月结日为 15 日
2. 预算周期将与账单周期一致
3. 更容易控制每个账单周期的支出

## 3. 预算滚存

### 什么是预算滚存？

未使用的预算金额可以累积到下个月。

### 滚存计算

```
本月预算 = 设置金额 + 上月剩余

示例：
- 月预算：$1000
- 上月剩余：$200
- 本月可用：$1200
```

### 启用滚存

1. 创建/编辑预算
2. 开启「启用滚存」选项

### 滚存注意事项

- 超支金额会从下月预算中扣除
- 滚存金额在预算详情中显示
- 可随时关闭滚存（从下月开始生效）

## 4. 预算追踪

### 进度显示

预算进度条颜色：

| 使用比例 | 颜色 | 状态 |
|---------|------|------|
| 0-70% | 绿色 | 正常 |
| 70-90% | 黄色 | 警告 |
| 90-100% | 橙色 | 接近超支 |
| >100% | 红色 | 已超支 |

### Dashboard Widget

预算进度 Widget 显示：
- 各分类预算使用情况
- 进度条可视化
- 剩余金额/超支金额

### 预算报告

在 Budgets 标签页查看：
- 当月所有预算汇总
- 历史预算对比
- 趋势分析

## 最佳实践

### 设置预算的建议

1. **从追踪开始** - 先记录一个月支出，了解消费模式
2. **设置实际目标** - 预算略低于平均消费
3. **逐步调整** - 每月根据实际情况微调
4. **关注大头** - 优先为大额支出分类设置预算

### 避免超支技巧

1. 启用超支提醒
2. 定期检查预算进度
3. 使用快速添加 Widget 即时记录
4. 月中检视调整消费
